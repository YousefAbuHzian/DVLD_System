# DVLD System  
Driver & Vehicle Licensing Department Management System

A full desktop application built with **C# (WinForms)** and **SQL Server**, designed to simulate and manage the daily operations of a driving license department.  
It allows administrators to handle license issuance, renewals, replacements, drivers, vehicles, and test management efficiently.
